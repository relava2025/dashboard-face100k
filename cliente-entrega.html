Como usar o Dashboard de Entrega
Siga estes 4 passos toda semana para usar o sistema corretamente.
1
Acesse sempre pelo Painel de Entrega
Nunca salve o link direto de um cliente. Entre pelo painel principal e clique no cliente para abrir o dashboard correto.
2
Veja "O que fazer hoje" na tela inicial
O card no topo da Visão Geral mostra as tarefas pendentes da semana atual. Comece por ele toda vez que abrir o sistema.
3
Marque as tarefas conforme concluir
Clique no checkbox ao lado de cada tarefa para marcá-la como feita. O sistema salva automaticamente.
4
Verifique os Alertas quando aparecerem
Se o botão "Alertas" aparecer em vermelho na barra lateral, há tarefas atrasadas — resolva antes de avançar para a próxima semana.
Clique no ? na barra superior para rever este guia
